# 🚀 Quick Start Guide - NutriScan App

Get the app running in 3 minutes!

## Step 1: Install Dependencies
```bash
npm install
```

## Step 2: Start Development Server
```bash
npm start
```

The app opens automatically at `http://localhost:3000`

## Step 3: Start Using!
- **Scan food** - Click the tab and select foods to log
- **View dashboard** - See your daily nutrition totals
- **Get suggestions** - Check personalized recommendations
- **View journal** - See your complete meal history

## Troubleshoot

| Problem | Solution |
|---------|----------|
| App won't start | Run `npm install` again |
| Port 3000 in use | Kill process or use different port |
| Styles broken | Restart dev server with `npm start` |
